# E-Commerce Frontend (React)

A responsive e-commerce frontend built using React.js, focused on clean UI, performance, and reusable components. Backend API consumed from a Django REST backend.

## Features
- Product listing with filters and sorting
- Product details page
- Cart management using Context API
- Authentication UI
- Responsive design

## Tech Stack
- React.js
- JavaScript (ES6+)
- CSS
- REST API integration

## Getting Started
1. Clone the repository  
2. Run `npm install`  
3. Run `npm start`

## Status
Frontend completed and actively maintained.
